# Node.js JSON API (Node puro)

Projeto de estudo para entender como funciona uma API HTTP sem frameworks.

## O que esse projeto faz
- Servidor HTTP usando Node.js puro
- CRUD de players
- Persistência em arquivo JSON
- Separação de responsabilidades
- Status HTTP e validação básica

## Tecnologias
- Node.js
- HTTP nativo
- fs
- JSON

## Objetivo
Aprender os fundamentos de backend antes de utilizar frameworks como Express ou Fastify.

## Como rodar
```bash
node server.js
A API ficará disponível em:
http://localhost:3000
